# Maturita angličtina — interaktivní tahák & kartičky

Interaktivní studijní pomůcka pro přípravu na ústní maturitu z angličtiny (úroveň B1–B2, obor Podnikání).

## Funkce

- **20 maturitních témat** s vzorovými otázkami a odpověďmi (Part A + Part B)
- **294 oficiálních Cermat otázek** s vzorovými odpověďmi (šablonové, s `*(placeholders)*` k doplnění)
- **1 222 flashcard karet** s češtinou, výslovností a anglickým překladem
- **Spaced repetition** — karty se chytře opakují podle tvého hodnocení
- **Cloud synchronizace** přes Supabase — pokrok se ukládá online, sdílení mezi zařízeními
- **Sub-záložky** podle struktury zkoušky: Otázky (Část A), Obrázky (Část 2), Konverzace (Část 3), Odborné (Část B), Slovní zásoba
- **Dark mode**, vyhledávání, export do AnkiWebu

## Použití

Otevři `index.html` v prohlížeči nebo navštiv [GitHub Pages odkaz](https://YOUR-USERNAME.github.io/maturita-anglictina/).

Při prvním otevření zadáš jednoduchou přezdívku — všechna zařízení se stejnou přezdívkou sdílí pokrok.

## Technologie

- Čisté HTML + JavaScript (žádný build proces)
- [Supabase](https://supabase.com/) jako backend pro cloud sync
- Hostováno přes GitHub Pages
